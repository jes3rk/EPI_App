API Experience:

One of my projects, CongressWatch makes use of ProPublica API. For this project, not only did I use the API to gather data and visualize it with D3, but I also built a server-side search engine for members of Congress, as searching by name wasn't supported by the API. The project had a number of challenges but was very rewarding in the long run.

In a similar vein, my work on Votify made use of D3 to visualize voting data from the state of Washington for the 2016 general election. This project gave me an excellent chance to work with real-world data and to create useful visualizations.
